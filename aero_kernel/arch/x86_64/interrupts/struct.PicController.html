Struct aero_kernel::arch::x86_64::interrupts::PicController

pub struct PicController;
Expand description

PIC (Programmable Interrupt Controller) manages hardware interrupts and sends them to the appropriate system interrupt for the x86 architecture. Since APIC has replaced PIC on modern systems, Aero disables PIC when APIC is available.

Notes

  • https://wiki.osdev.org/8259_PIC
  • https://wiki.osdev.org/APIC

unsafe fn get_irq_register(&self, command: u8) -> u16

Helper function to get the IRQ register. This function is responsible for sending the provided command PIC master to get the register values. PIC master represents IRQs 0-7, with 2 being the chain. PIC slave is chained, and represents IRQs 8-15.

pub type mask16x64 = Mask<i16, 64>;
🔬This is a nightly-only experimental API. (portable_simd)
Expand description

A SIMD mask with 64 elements for vectors with 16-bit element types.

The layout of this type is unspecified, and may change between platforms and/or Rust versions, and code should not assume that it is equivalent to [i16; 64].

🔬This is a nightly-only experimental API. (portable_simd)

Choose lanes from two vectors.

For each lane in the mask, choose the corresponding lane from true_values if that lane mask is true, and false_values if that lane mask is false.

Examples
let a = Simd::from_array([0, 1, 2, 3]);
let b = Simd::from_array([4, 5, 6, 7]);
let mask = Mask::from_array([true, false, false, true]);
let c = mask.select(a, b);
assert_eq!(c.to_array(), [0, 5, 6, 3]);

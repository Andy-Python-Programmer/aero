Expand description

Scheduler queue containing a vector of all of the task of the enqueued taskes.

Fields§

§idle_task: Arc<Task>

The kernel idle task is a special kind of task that is run when no taskes in the scheduler’s queue are available to execute. The idle task is to be created for each CPU.

Copy to/from a block of data from user space. Returns whether the copy was successful.

§Safety

The caller must ensure that:

  • If copying to userspace, the dest pointer must be within the userland address space.
  • If copying from userspace, the dest pointer must be valid for size bytes.

§Concurrent Accesses

Concurrent access, including potential data races with userspace memory, are permitted since other userspace threads or processes may modify the memory concurrently. This behavior is similar to how std::io permits data races with file contents on disk.

64-bit SYSENTER instruction entry point.

The SYSENTER mechanism performs a fast transition to the kernel.

The new CS is loaded from the IA32_SYSENTER_CS MSR, and the new instruction and stack pointers are loaded from IA32_SYSENTER_EIP and IA32_SYSENTER_ESP, respectively. RFLAGS.IF is cleared, but other flags are unchanged.

As the instruction does not save any state, the user is required to provide the return RIP and RSP in the RCX and R11 registers, respectively. These addresses must be canonical.

The instruction expects the call number and arguments in the same registers as for SYSCALL.

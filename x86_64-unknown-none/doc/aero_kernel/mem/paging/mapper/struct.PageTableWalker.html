fn create_next_table<'b>( &self, entry: &'b mut PageTableEntry, insert_flags: PageTableFlags, ) -> Result<(bool, &'b mut PageTable), PageTableCreateError>

Internal helper function to create the page table of the next level if needed.

If the passed entry is unused, a new frame is allocated from the given allocator, zeroed, and the entry is updated to that address. If the passed entry is already mapped, the next table is returned directly.

Returns MapToError::FrameAllocationFailed if the entry is unused and the allocator returned None. Returns MapToError::ParentEntryHugePage if the HUGE_PAGE flag is set in the passed entry.

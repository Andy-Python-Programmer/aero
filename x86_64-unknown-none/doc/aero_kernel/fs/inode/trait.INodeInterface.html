fn rename( &self, _src: CacheArc<CacheItem<(usize, String), DirEntry>>, _dest: &str, ) -> Result<(), FileSystemError>

Renames a file (src) to dest, moving it between directories if required. Any other hard links to the file (as created using INodeInterface::link) are unaffected.

ยงAtomicity
  • If dest already exists, it will be atomically replaced.
  • There can be a window in which both src and dest refer to the file being renamed.

Struct SmpResponse

#[repr(C)]
pub struct SmpResponse { /* private fields */ }
Expand description

A response to a smp request. This response contains information about the boot processor and all other processors.

pub fn cpus_mut(&mut self) -> &mut [&mut Cpu]

Returns a mutable slice of found CPUs. See [Cpu] for more information. Note that this function takes &mut self, so the response will likely need to be wrapped in a Mutex or similar. It is provided so that the extra field on each CPU can be set.
